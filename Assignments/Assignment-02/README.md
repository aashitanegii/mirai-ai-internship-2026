# ⚽ Football Legends AI Chatbot

An AI chatbot built using Streamlit and Google Gemini.

Choose your favourite football legend and ask them anything.

## Run locally

1. Create a .env file from .env.example.
2. Add your Gemini API key.
3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the app:

```bash
streamlit run app.py
```
